Colorflow

A simple-to-understand and useful gradient, or color previewer and generator.

Why? • I love working with CSS colors, so I wanted to create myself a tool that allowed me to see different CSS colors in different scenarios. I also wanted to use it to test other people's colors I found online if they didn't show how it looked.
